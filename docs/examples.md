# Examples

This page collects simple examples for common AegisLab tasks.

## Example Scenarios

- Submit a fault injection and inspect task state transitions.
- Build datapacks after injection completion.
- Execute RCA algorithms and compare evaluation results.

## Suggested Validation Path

Start with a minimal local scenario, then scale to cluster workloads once the flow is stable.
