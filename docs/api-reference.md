# API Reference

AegisLab exposes REST APIs for auth, projects, datasets, injections, executions, evaluations, and metrics.

## API Surface

The active implementation in this repository is primarily under versioned routes such as `/api/v2`.

## How to Inspect Endpoints

Use generated Swagger/OpenAPI documentation from source annotations to inspect request and response contracts.

## Domain Groups

- Authentication and authorization
- Project and label management
- Fault injections and datapacks
- Algorithm execution and evaluation
- Metrics and trace retrieval
