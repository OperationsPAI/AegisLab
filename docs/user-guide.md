# User Guide

This guide provides a practical path for using AegisLab (RCABench) in daily workflows.

## Typical Workflow

1. Start required dependencies and backend services.
2. Submit fault injection requests through API or SDK.
3. Track task progress and telemetry.
4. Build datapacks and run RCA algorithms.
5. Compare evaluation metrics across runs.
